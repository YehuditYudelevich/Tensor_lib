# Tensor Library in C

A lightweight C library for working with multi-dimensional `float` arrays.

The goal is to provide a simple foundation for numerical computing in C. A tensor can represent a vector, matrix, or higher-dimensional array, and can be used as a base for future machine-learning or scientific-computing projects.

The final library will support creating tensors, accessing values, reshaping, transposing, slicing, broadcasting, and element-wise arithmetic such as addition, subtraction, multiplication, and division.
